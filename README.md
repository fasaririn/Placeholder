## 💻 <a id="how-do-i-run-it"></a>How do I run it?

If you are a judge or a fellow developer who wants to run the game locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the project folder in your terminal or <kbd>VS Code</kbd>.
3. Run the provided build scripts. 

You can execute the scripts directly from your terminal:

```bash
# For Windows Users:
build.bat

# For Mac/Linux Users:
./build.sh
```

4. Finally, open `index.html` or `game.html` in your preferred web browser.

## 🛠️ <a id="how-to-customize"></a>How to Customize (Code Example)

Because we wanted to make game development accessible, you don't need to write complex logic to add a new character or scene. The game is driven entirely by `JSON` data!

Here is an example of how easy it is to define a character in our `data/characters.json` file:
```json
{
  "characters": [
    {
      "id": "hero_01",
      "display_name": "Alex",
      "text_color": "#FF5733",
      "sprite_image": "img/alex_neutral.png"
    }
  ]
}
```
*Just swap the text and the image, and a brand-new character appears in the game!*
